# 路线图

## v0.1：JSON 地图求解

建立不可变领域模型、JSON 校验、配置驱动移动规则、记忆化精确求解、CLI、样例与自动测试。本仓库当前只完成此阶段。

## v0.2：截图文件识别

在不控制设备的前提下，读取用户提供的截图文件，探索节点定位、OCR、模板匹配、置信度表达和人工校正接口。识别层输出 v0.1 的领域模型。

## v0.3：MaaFramework 接入

通过明确适配层调用 MaaFramework 的截图、OCR、模板匹配和设备能力；遵守上游许可及接口约束，不复制 MAA 代码或图片资源。

## v0.4：独立可视化界面

提供地图校正、候选路线解释、资源设置和模拟展示。UI 不承载求解规则。

## v1.0：动态重规划和 Windows 发布

根据每一步执行后的新观测更新状态并重新规划，完善异常恢复、打包、版本升级和 Windows 发布流程。任何自动控制能力都需要单独的安全设计与用户授权。
